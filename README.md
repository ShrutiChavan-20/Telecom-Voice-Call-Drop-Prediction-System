## рџ“– Overview

Telecommunication companies receive thousands of customer complaints regarding poor signal quality and frequent call drops. Traditional monitoring systems often identify issues only after they occur.

This project leverages Machine Learning to predict the probability of call drops based on telecom complaint data, enabling proactive network analysis and better decision-making through interactive dashboards.
